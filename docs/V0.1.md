# Gas Surfer v0.1 — User guide & release notes

## What’s in v0.1

- **Bitcoin** — Featured card with Slow / Standard / Fast in sat/vB (from mempool.space). Hint: “~1 hr · ~30 min · next block”.
- **Ethereum** — Featured card with gwei tiers and fiat cost for a 21k gas tx.
- **13 EVM chains** — Base, Arbitrum One, Optimism, Polygon, BNB Smart Chain, Avalanche C-Chain, Fantom, Linea, Gnosis, zkSync Era, Mantle, Celo. Each shows Slow / Standard / Fast (gwei) and approximate fiat cost.
- **Currency selector** — USD, EUR, GBP, JPY, CHF, CAD, AUD. All “cost per tx” and chart values use the selected currency where applicable.
- **Surf report** — Hero section for the currently selected chain: condition (Surf’s up / Smooth / Choppy / Storm), standard fee, and 7/30/90/180-day averages when available.
- **Fee averages** — Stored in `localStorage` per chain; shown on each card and in the surf report.
- **Mini chart** — Last N updates of the standard fee for the selected chain.
- **Safe for static hosting** — No API keys; all endpoints in config; CSP and referrer policy in `index.html`.

---

## How to use

1. **Open the app** — Dev: `npm run dev` → http://localhost:5173. Production: serve `dist/` after `npm run build`.
2. **Pick a chain** — Click Bitcoin, Ethereum, or any EVM chain card. The surf report and chart update for that chain.
3. **Change currency** — Use the currency dropdown; all fiat amounts and cost estimates update.
4. **Read the tiers** — Slow / Standard / Fast are the three fee levels. For Bitcoin, the small hint under the numbers explains ~1 hr / ~30 min / next block.

---

## Surf conditions

| Condition   | Meaning (rough)        |
|------------|-------------------------|
| Surf's up  | Fees low; good time to send |
| Smooth     | Moderate fees           |
| Choppy     | Higher fees             |
| Storm      | High fees               |

Thresholds are per-chain (e.g. Bitcoin uses sat/vB bands; EVM uses gwei bands and differs for L1 vs L2).

---

## Data and privacy

- Gas/fee data: public RPCs and mempool.space. No account, no keys.
- Prices: CoinGecko public API.
- Averages: computed and stored only in your browser (`localStorage`). Nothing is sent to our own servers (there are none).

---

## Known limitations (v0.1)

- **Public RPCs** — Rate limits and availability vary; the app uses several fallbacks per chain.
- **Bitcoin** — Only mainnet; no Lightning.
- **Averages** — Based on the standard fee at each refresh; stored per chain in the client only.
- **No alerts** — No notifications or “surf’s up” alerts; manual check only.
